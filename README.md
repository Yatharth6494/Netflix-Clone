# Netflix Signup Clone (Static UI)

A project focused on recreating the signature Netflix landing and signup page using only **HTML** and **CSS**.

## 🚀 Project Overview
This is a front-end exercise to practice pixel-perfect design and CSS layouts. The goal was to replicate the visual aesthetic of the Netflix brand, specifically focusing on the landing page and signup form.

### 🛠️ Built With
* **HTML5:** Semantic structure.
* **CSS3:** Custom styling (Flexbox and Background layering).

## 🚧 Status & Limitations
Please note that this is a **Static UI Study**:
* **Non-Responsive:** Currently optimized for desktop screens only.
* **Static Form:** The "Get Started" button and email input are visual only and do not process data.
* **Single Page:** This project focuses solely on the Signup/Landing UI.

## 📖 What I Learned
During this project, I focused on:
* Using **Linear Gradients** to create the "overlay" effect on background images.
* Mastering **Flexbox** for centering the main call-to-action.
* Customizing **Form Inputs** to match specific brand guidelines.
